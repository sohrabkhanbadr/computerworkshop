عناصر پایه HTML

در اینجا می‌توانید عناصر پایه HTML را یاد بگیرید:

  • سرتیترها (Headings)
  • پاراگراف‌ها (Paragraphs)
  • لینک‌ها (Links)
  • تصاویر (Images)

سرتیترها (Headings)

سرتیترها با تگ‌های <h1> تا <h6> تعریف می‌شوند:
