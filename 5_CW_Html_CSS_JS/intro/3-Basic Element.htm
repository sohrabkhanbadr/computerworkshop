عناصر پایه HTML

مقدمه

HTML (HyperText Markup Language) زبانی است که برای ساخت صفحات وب استفاده می‌شود. صفحات وب شامل متن، تصاویر، ویدیو و سایر عناصر هستند که با استفاده از تگ‌های HTML تعریف می‌شوند. تگ‌ها عناصر HTML را مشخص می‌کنند و به مرورگرها می‌گویند که چگونه آنها را نمایش دهند.

ساختار سند HTML

هر سند HTML با اعلان نوع سند (Document Type Declaration) شروع می‌شود:

سرتیترها

سرتیترها برای دسته‌بندی و سازماندهی محتوای صفحه استفاده می‌شوند. سرتیترها با تگ‌های <h1> تا <h6> تعریف می‌شوند. <h1> مهم‌ترین سرتیتر را نشان می‌دهد و <h6> کم‌اهمیت‌ترین آن را مشخص می‌کند.

        
این سرتیتر 1 است

        
این سرتیتر 2 است

        
این سرتیتر 3 است

پاراگراف‌ها

پاراگراف‌ها بلوک‌های متنی هستند. پاراگراف‌ها با تگ <p> تعریف می‌شوند.

        
این یک پاراگراف است.

        
این پاراگراف دیگری است.

لینک‌ها

لینک‌ها به کاربران اجازه می‌دهند تا بین صفحات وب مختلف حرکت کنند. لینک‌ها با تگ <a> تعریف می‌شوند. ویژگی href مقصد لینک را مشخص می‌کند.

        <a href="https://www.w3schools.com">این یک لینک است</a>

تصاویر

تصاویر با تگ <img> تعریف می‌شوند. ویژگی `src` مسیر فایل تصویر را مشخص می‌کند. ویژگی‌های `alt`، `width` و `height` به ترتیب متن جایگزین، عرض و ارتفاع تصویر را مشخص می‌کنند.

        <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">

نکات مهم

  • تگ‌های HTML باید به صورت جفت استفاده شوند.
  • تگ‌های HTML باید به حروف کوچک و بزرگ نوشته شوند.
  • تگ‌های HTML باید به ترتیب صحیح قرار بگیرند.
  • می‌توانید از ابزارهای مختلفی مانند ویرایشگرهای HTML برای نوشتن و ویرایش کد HTML استفاده کنید.
